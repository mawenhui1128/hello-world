1，拉取git

cd /home/xiaoju
git clone git@git.xiaojukeji.com:gs-qa/dws.git
git clone git@git.xiaojukeji.com:dongzongqing/py_demo.git    ps：这是部署py_demo的环境；如果部署自己产品线的环境，请拉取自己的代码。

2，各种设置

设置软连接

cd /home/xiaoju/dws
ln -s /home/xiaoju/py_demo/doc doc

设置dws

vi ~/.bashrc，在路径中添加/home/xiaoju/dws，比如
export PATH=/home/xiaoju/php7/bin:$PATH:/home/xiaoju/dws
source ~/.bashrc

把以下文件复制到/home/xiaoju路径下

/home/xiaoju/dws/profile/py_demo/environment.json   用于设置http服务器、mysql、redis等各种地址

/home/xiaoju/dws/profile/py_demo/db.conf   用于设置mysql的用户名和密码。（py_demo无该文件。如果你的产品线有该文件的话，务必拷贝）

这就可以了，可以执行dws test --include=abc指令了。如果不可以，说明python环境有问题，需要安装各种lib。

根据运行时候的报错，安装所需的lib（不一定需要全部安装）。

安装这些lib，可能需要外网权限，请开通。

一直到运行dws test --include=abc不报错了，就可以了。

 

1，查看python版本

python --version
如果版本低于2.7.11，请升级

2，安装python

下载https://www.python.org/ftp/python/2.7.13/Python-2.7.13.tgz
tar zxvf Python-2.7.13.tgz
./configure --prefix=/usr/local/python
make
make install
mv /usr/bin/python /usr/bin/python2.7
ln -s /usr/local/python/bin/python2.7 /usr/bin/python
python --version, 检查确认python安装成功
注，升级后yum不能提示版本不对不能使用，修改/usr/bin/yum 将文件头部#!/usr/bin/python 修改为 /usr/bin/python2.7 (之前python的版本)，即可使用

3,安装pip

wget --no-check-certificate https://bootstrap.pypa.io/ez_setup.py
sudo python ez_setup.py --insecure
wget https://pypi.python.org/packages/11/b6/abcb525026a4be042b486df43905d6893fb04f05aac21c32c638e939e447/pip-9.0.1.tar.gz#md5=35f01da33009719497f01a4ba69d63c9
tar -xf pip-9.0.1.tar.gz
cd pip-8.0.0
sudo python setup.py install
rm /usr/bin/pip
ln -s /usr/local/python/bin/pip /usr/bin/pip
或者wget https://bootstrap.pypa.io/get-pip.py
python get-pip.python

4,安装gevent

pip install gevent

5,安装MySQLdb

pip install MySQL-python 
安装错误
yum -y install mysql-devel 
提示缺少libmysqlclient.so.18
执行yum install mariadb-libs
若与本机mysql冲突，先卸载本机mysql
rpm -e mysql*

6，安装protobuf

pip install protobuf

7，安装requests

pip install requests

8，安装ply

pip install ply

9，安装devenv

pip install devenv
